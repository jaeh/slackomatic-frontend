▶ Video ouput doesn't change on the projector

Please be patient, this problem occurs because the projector sometimes needs some "think time" to decide what to do next. Just press the button for the system again after a few seconds.

▶ Yamaha doesn't change input

The Yamaha web interface has some stability problems (request just time out). Please power toggle the Yamaha using the physical button on the device itself. After standby/on the interface was available again last time.

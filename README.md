slackomatic frontend
====

To install:
```bash
git clone https://github.com/jaeh/slackomatic-frontend 
cd slackomatic-frontend
bin/install
```

To run:
```bash
cd slackomatic-frontend
npm start
```

To start on boot in /etc/inittab on raspbian 
with the source in /home/pi/nodejs:
```bash
cd /home/pi/nodejs/ && npm start
```
